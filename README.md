# 📦 Backend CRUD App

A simple backend application implementing full CRUD operations using **Node.js**, **Express**, and **MongoDB**. 

---
## 🔧 Features

- ✅ Create, Read, Update, and Delete (CRUD) functionality
- 💾 MongoDB with Mongoose

## ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemon (for development)
